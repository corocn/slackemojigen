# Slack Emoji Generator

Slack Emoji Generator is blazing fast emoji generator for Slack

https://slackemojigen.com/
 
## Setup

```
yarn install
yarn dev
```

## Supported browsers

 * Chrome
 * Firefox
 * Safari
